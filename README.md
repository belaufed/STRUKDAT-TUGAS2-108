### Problem

Dalam proses produksi lagu sederhana, sering terjadi kebingungan dalam mencatat informasi seperti siapa saja yang terlibat dalam pembuatan lagu serta detail lagu yang sedang dikerjakan. Misalnya, dalam satu tim terdapat beberapa peran seperti producer dan rapper, namun pencatatan masih dilakukan secara manual atau tidak terstruktur sehingga informasi mudah tercampur atau terlupakan.

Oleh karena itu, dibuat sebuah program sederhana berbasis konsep Object-Oriented Programming (OOP) untuk membantu mengelola data tersebut. Program ini memungkinkan pengguna untuk merepresentasikan setiap individu berdasarkan perannya serta menyimpan informasi lagu yang diproduksi, sehingga data menjadi lebih terorganisir dan mudah ditampilkan kembali saat dibutuhkan.
